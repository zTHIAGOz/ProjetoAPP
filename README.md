# ProjetoAPP
Projeto pessoal que fiz pra uso prático. Tanto para mostrar conhecimento em Java, quanto para treinar e aplicar os mesmos. É uma lista de tarefas, inicialmente, com foco em se tornar um app que ajudará pessoas a se organizar com rotinas e, futuramente, na vida financeira e com seus estudos (pretendo colocar pomodoro, rotina de estudos, mapa mental)
